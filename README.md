# Contest Code Site

## Goal
A responsive website where users enter a 3-digit code.
- Max 3 attempts
- 10-minute lock after 3 failed attempts
- Only one winner total
- Only winner can submit contact details
- Admin can reset contest (test mode)

## Tech
- Frontend: React + TypeScript (planned)
- Backend: Python (FastAPI)
- Database: PostgreSQL
